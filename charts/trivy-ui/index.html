About Trivy UI

Trivy UI is a web-based dashboard for visualizing and managing Trivy vulnerability scan results across multiple Kubernetes clusters. This Helm chart provides a complete deployment solution with support for multi-cluster management, RBAC integration, and flexible configuration options.

# Add the Helm repository helm repo add trivy-ui https://locustbaby.github.io/trivy-ui/ helm repo update # Install the chart helm install my-trivy-ui trivy-ui/trivy-ui

From Docker Hub

# Install directly from Docker Hub Helm registry helm install my-trivy-ui oci://registry-1.docker.io/locustbaby/trivy-ui
